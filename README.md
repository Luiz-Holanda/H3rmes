# H3rmes Logística React App

Este projeto é uma transformação de arquivos HTML estáticos em uma aplicação React moderna, performática e totalmente estruturada com TypeScript.

## 🚀 Tecnologias Utilizadas

- **React 18** com **TypeScript**
- **Vite** para build e desenvolvimento rápido
- **Tailwind CSS** para estilização
- **React Router Dom** para roteamento
- **Framer Motion** para animações
- **Vitest** e **React Testing Library** para testes unitários
- **ESLint** e **Prettier** para padronização de código

## 📂 Estrutura de Pastas

```text
src/
├── assets/      # Imagens e recursos estáticos
├── components/  # Componentes reutilizáveis (Header, Footer, Layout)
├── hooks/       # Hooks customizados
├── pages/       # Páginas da aplicação (Home, Dashboard, Login, etc.)
├── services/    # Chamadas de API e serviços externos
├── styles/      # Arquivos de estilo globais
├── test/        # Configuração de testes e testes unitários
├── types/       # Definições de tipos TypeScript
└── utils/       # Funções utilitárias
```

## 🛠️ Instalação e Execução

Para rodar o projeto localmente, siga os passos abaixo:

1. **Instalar dependências:**
   ```bash
   npm install
   ```

2. **Iniciar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Rodar os testes:**
   ```bash
   npm run test
   ```

4. **Rodar o lint:**
   ```bash
   npm run lint
   ```

5. **Gerar build de produção:**
   ```bash
   npm run build
   ```

## ✨ Funcionalidades Implementadas

- **Home Page:** Seção Hero, calculadora de frete dinâmica e grade de serviços.
- **Dashboard:** Painel de controle do usuário com encomendas ativas e histórico.
- **Rastreamento:** Busca de rastreio detalhada com linha do tempo e mapa.
- **Suporte:** Central de ajuda com FAQ interativo e métodos de contato.
- **Login/Cadastro:** Interface de autenticação moderna com suporte para Pessoa Física e Jurídica.
- **Design Responsivo:** Totalmente adaptado para dispositivos móveis e desktops.
